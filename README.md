# cygnomatic

The site is generated using [vitepress](https://vitepress.dev/)

## Install

```sh
pnpm i
```

## Usage

```sh
# start dev server
pnpm docs:dev
# build for production
pnpm docs:build
# preview production build in local
pnpm docs:preview
```
